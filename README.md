## **CaptialMangementSystem资金管理系统**
### 1简介
资金管理系统是基于Express框架、Vue和Element-UI联合开发的管理系统，结合了当前前端许多主流技术，让我们实现全栈的贯通，非常有学习的价值。

**运行方式**
1. 在client目录下加入依赖

```javascript
npm insatll
```

 2. 在项目根目录下加入依赖
 
```javascript
npm insatll
```
 3. 在项目目录下打开终端，输入以下代码
```javascript
nodemon server.js
```
 4. 在client目录下打开终端，输入以下代码
```javascript
npm run serve
```
